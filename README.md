🩺 AI-Powered Medical Image Analysis System
AI-powered medical image analysis system for pneumonia detection using deep learning and Streamlit web app


📌 Overview
This project uses Deep Learning to detect Pneumonia from Chest X-ray images.

 🚀 Features
- Image classification using MobileNetV2
- 97–99% accuracy
- Streamlit web app for real-time prediction
- Visualization (accuracy graph, confusion matrix)

🧠 Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Streamlit

 📂 Dataset
Chest X-ray dataset (public)
